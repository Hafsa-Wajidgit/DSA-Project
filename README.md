# SocialMediaManagement
## Social Media Feed Management System
A C++ based social media management system developed as a **Data Structures & Algorithms Lab final project**.  
The project demonstrates practical usage of **Binary Search Trees, Linked Lists, Heaps, and Stacks** to build a modular system.

## Modules
### Module I: User Management
- User registration and login
- Profile management
- Follow / unfollow system
- Persistent storage using files  
**Data Structures:** Binary Search Tree, Linked List

### Module II: Post Management
- Create, edit, and delete posts (280-character limit)
- Like / unlike posts
- Comment system
- Personalized feed from followed users  
**Data Structures:** Doubly Linked List, Singly Linked List

### Module III: Feed Generation & Notifications
- Priority-based notifications (Comments > Likes > Follows)
- Min-heap implementation for notifications
- Read / unread tracking
- Browsing history with back navigation  
**Data Structures:** Min-Heap, Stack

### Module IV: GUI Interface
- Graphical user interface using Dear ImGui
- Dark theme with modern layout
- Screens for login, feed, profile, search, post creation, and notifications
